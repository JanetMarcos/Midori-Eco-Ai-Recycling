Midori: AI-Powered Recycling Assistant
Description:
Midori is an AI-driven recycling assistant designed to promote sustainable waste management on the University of Salford campus. The application uses an interactive interface powered by Streamlit, combined with AI object recognition, to guide users in sorting waste properly. A webcam-enabled scanner and voice assistant enhance accessibility and help educate users on recycling best practices.

__________________________________________________________________________________
Features
AI-based object recognition using the mobilenet_iter_73000.caffemodel model.
Webcam scanner identifies recyclable items and guides bin placement.
Voice assistant for accessible and user-friendly interaction.
Educational feedback to improve recycling knowledge and habits.

Requirements
Python 3.x
Required libraries and dependencies listed in requirements.txt.
mobilenet_iter_73000.caffemodel (place in the project directory).
